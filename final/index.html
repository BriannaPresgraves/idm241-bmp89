Final Description

Scully and Scully

Triggers

  • Hover on/off product information tile
  • Hover on/off quick shop button
  • Hover on/off left/right arrows
  • Click left/right arrows
  • Click quickshop button
  • Hover on/off quantity buttons
  • Click quantity buttons
  • Hover on/off add to bag button
  • Hover on/off heart button
  • Hover on/off modal exit button
  • Click on modal exit button

Rules

Hover on product info tile

  • Quick shop button appears below product image
  • Left and Right arrows quickly fade in on the sides of the product image
  • Drop shadow fades in behind tile

Hover off product info tile

  • Quick shop button, arrows, and drop shadow quickly fade out

Hover on quick shop button

  • Button font color quickly fades to white

Hover off quick shop button

  • Font color quickly fades back to black

Hover on left/right arrows

  • Cursor changes to pointer

Hover off left/right arrows

  • Pointer changes back to cursor

Click left/right arrows

  • If the right arrow is clicked, current image slides to the left while the next image slides in from the right
  • If the left arrow is clicked, current image slides to the right while the next image slides in from the left

Click quickshop button

  • Modal with additional product info opens, expanding out from middle

Hover on quantity buttons

  • Cursor changes to pointer

Hover off quantity buttons

  • Pointer changes back to cursor

Click on quantity buttons

  • The quantity will increase or decrease
  • The quantity will be updated in the database

Hover on add to bag button

  • Button background color quickly fades to a darker blue

Hover off add to bag button

  • Button background color quickly fades back to original color

Hover on heart button

  • Heart slowly fades to pink color

Hover off heart button

  • Heart slowly fades back to white color

Hover on modal exit button

  • Cursor changes to pointer

Hover off modal exit button

  • Pointer changes back to cursor

Click modal exit button

  • Modal with additional product info quickly fades out and disappears
